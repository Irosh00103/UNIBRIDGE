# UniBridge

## Setup
1. cd server && npm install
2. Add your MongoDB URI to server/.env
3. npm run seed (creates test users and data)
4. npm run dev

5. cd client && npm install
6. npm start

## Test Accounts
- Student: student@test.com / test123
- Employer: employer@test.com / test123

## Features Built
- JWT Authentication with role-based routing
- Lecture Materials: upload (via link) + browse
- Job System: employer posts, student applies, employer reviews
- Kuppi Hub: create and view peer learning sessions
- Form validation on all forms
- Fill Sample Data buttons for fast demo
